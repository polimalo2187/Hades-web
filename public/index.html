Motor de estrategias

Dos lecturas. Una misma idea: no perseguir el mercado.

⚔

Liquidity Sweep

Lectura de barridas de liquidez: zonas donde el mercado limpia posiciones y puede buscar reacción. Usa filtros de sweep, volumen, confirmación, pullback, RR y avance hacia objetivos.

  • Barridas de liquidez
  • Confirmación operativa
  • Entrada anticipada con espera
  • Filtros endurecidos para reducir señales flojas
↗

Breakout + Reset

Detecta rupturas de estructura, evita perseguir el precio y espera una zona de reset o retroceso para una entrada más ordenada.

  • Breakout reciente
  • Reset estructural
  • Señal anticipada
  • Estado pendiente hasta tocar entrada
